## 🚀 About the Project

**ShubhDeals** is a modern **affiliate-based eCommerce web app** that aggregates top deals from platforms like **Amazon, Flipkart, and Meesho** into a single, clean interface.

It replicates a **real-world shopping experience** with cart, checkout, wishlist, and order system — built completely using **frontend technologies**.

---

## ✨ Key Features

### 🛒 Shopping Experience

* Add to Cart / Remove Items
* Quantity management
* Wishlist system ❤️

### 🔍 Product System

* Category filtering
* Product detail modal
* Ratings & discount display

### 💳 Checkout System

* Full form validation
* UPI & Cash on Delivery
* Order summary calculation

### 📦 Order Management

* Unique Order ID generation
* Order storage (localStorage)
* Confirmation screen

### 🛠️ Admin Panel (Hidden)

* Shortcut: `Ctrl + Shift + A`
* View all orders
* Configure EmailJS

### 🔥 Extra Features

* Flash sale countdown timer
* Toast notifications
* Affiliate redirect system
* Persistent cart & wishlist
